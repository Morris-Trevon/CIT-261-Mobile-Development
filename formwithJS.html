Before writing any program you must know the requirements. A simplified requirement can be will it have input and will it output anything after taking that input. Once you have ascertained these two things, you can then go into how we achieve those requirements.

Variables and Functions

In Javascript variables are containers or holding areas for data. You give variables special names so you can remember exactly what that variable holds. Before you can use a variable it must be declared using the var keyword. See example below:

Declaring a variable

var age;
var firstName;
var lastName;

You can declare a variable with a value which is called initializing:

var age=0;
var firstName=0;
var lastName=0;

When naming variables in Javascript a. The first character must be a letter or an underscore (_) b. The rest of the variable name can contain any letter or number. c. variable names are case sensitive, so FIRST_NAME is different from first_name. d. Do not use Javascript keywords like break or boolean.

So ensure you have a standard naming methodology for variables.

Literal Data Types

Javascript variable initializing is simple compared to most programming languages. The first data type to enter that variable will decide the variables literal data type. This is important to note for you may want to add two numbers, so both variables must be of the same data type.

Common data types are: integer, floating point, and text/ string

Functions and Parameters

A function in Javascript is a segment of code reserved for a specified purpose or a number of related purposes. A function must be called or invoked by events, code, or automated. Functions are defined with the function keyword, followed by the name of the function, parameter parantheses (), and then enclose the code in curly braces {}. Function names follow the same rules as variables. See example below:

function submitData() {
//code to be executed
return false;
// return statement exits the function

}

If you notice the return statement that will exit the function. If the statement above said true then the function is expected to send data to wherever it is called from. Functions can be called by a variable to return data into that variable eg. firstName=getFirstName(); in that example the return statement would like return firstName;

Functions are reusable and can accept and return values to different sources.

Conditional Statement

Condition statements are used to run a block of code once. You can embed anything within a conditional statement as long as you want that code to run once. There are 3 types of conditional statements 1. if statement runs the code once based on the condition, 2. if else statement runs one of two scenarios one statment if true and the other if false 3. switch statement runs through a number of scenario to see if a value matches. You can also nest if else statments to run multiple scenarios that will execute only once this is called else if.

if statement

if (condition) {
  //  block of code to be executed if the condition is true
}

if else statement

if (condition) {
  //  block of code to be executed if the condition is true
} else {
  //  block of code to be executed if the condition is false
}

else if statment / nested if statement

if (condition1) {
  //  block of code to be executed if condition1 is true
} else if (condition2) {
  //  block of code to be executed if the condition1 is false and condition2 is true
} else {
  //  block of code to be executed if the condition1 is false and condition2 is false
}

switch statement

switch(expression) {
  case x:
    // code block
    break;
  case y:
    // code block
    break;
  default:
    // code block
}

Arrays and Associative Array

Arrays are special variable or objects used to store multiple values of the same data type. Example below:

var cars = ["Saab", "Volvo", "BMW"];

Array values can be referenced and changed based on their address called the index. which starts from 0.

var favCar = cars[0]; // This would assign Saab into the variable called favCar

document.getElementById("demo").innerHTML = cars[0]; // This would output Saab to the page

You can also access the full array by simply outputting the array name with no address.

Arrays can do mathematical calculations against itself and store its answer within itself. They are very flexible and can be used as a method of temporary storage for multiple values

Associative Arrays

These arrays are special in that you can name the index, but they are not supported in Javascript. Javascript only supports numbered index.

person["firstName"] = "John";

Javascript will not retain the name and revert to numbered index.
