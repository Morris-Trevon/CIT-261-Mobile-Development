Before writing any program you must know the requirements. A simplified requirement can be will it have input and will it output anything after taking that input. Once you have ascertained these two things, you can then go into how we achieve those requirements.

Variables and Functions

In Javascript variables are containers or holding areas for data. You give variables special names so you can remember exactly what that variable holds. Before you can use a variable it must be declared using the var keyword. See example below:

Declaring a variable

var age;
var firstName;
var lastName;

You can declare a variable with a value which is called initializing:

var age=0;
var firstName=0;
var lastName=0;

When naming variables in Javascript a. The first character must be a letter or an underscore (_) b. The rest of the variable name can contain any letter or number. c. variable names are case sensitive, so FIRST_NAME is different from first_name. d. Do not use Javascript keywords like break or boolean.

So ensure you have a standard naming methodology for variables.

Literal Data Types

Javascript variable initializing is simple compared to most programming languages. The first data type to enter that variable will decide the variables literal data type. This is important to note for you may want to add two numbers, so both variables must be of the same data type.

Common data types are: integer, floating point, and text/ string

Functions and Parameters

A function in Javascript is a segment of code reserved for a specified purpose or a number of related purposes. A function must be called or invoked by events, code, or automated. Functions are defined with the function keyword, followed by the name of the function, parameter parantheses (), and then enclose the code in curly braces {}.See example below:

function submitCatch() {
//code to be executed
return false;

}
