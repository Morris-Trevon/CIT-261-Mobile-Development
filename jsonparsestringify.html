JSON Parse, Stringify

This page demonstrates fluency in the above topics.

JSON stands for JavaScript Object Notation. It is pure text and we can convert any JavaScript object to text. Information between Server and browser is text, so JSON can also be turned from text to objects. So JavaScript can work through the browser with no issue and worry about parsing and translations.

Using JavaScript to read JSON object

function generateJSON(){

//JSON code

data = {Movie: "Golden Eye", releaseDate: "Nov 13, 1995", budget: "$60 million", boxoffice:"$355 million"};

//Declare variable and load JSON string into it

var stringJSON = JSON.stringify(data); //Converts the JSON data to a string

//parse the Stringify object and turn each variable of the JSON into a referencable object
var movie = JSON.parse(stringJSON);

//Output any portion of the JSON code by calling movie.[nameofobj]
document.getElementById("output").innerHTML = "Movie Name: "+movie.Movie+"<br>Release Date: "+movie.releaseDate+"<br>Budget: "+movie.budget+"<br>Box Office: "+movie.boxoffice;

}

JSON can be used to create any type of object with JavaScript and then sent over the browser and displayed in many different forms.
